Complex Note App

This note app is designed for the storage and retrieval of notes meant for specific scenarios, such as matchups within competitive gaming. You are also able to further filter the notes based on specified data, so as to make them easier to read.

This app was built using HTML, CSS, Javascript and jQuery. The data is pulled from a prototype database made in Google Sheets, using the Tabletop.js library.

Bikeshare App

This bikeshare was tailored with mobile usage in mind, and searches for available bikes as well as empty slots to dock your rented bike in. It also finds the closest dock to your location and gives real-time directions to it.

This app was made in HTML, CSS, Javascript and jQuery. The data is pulled from the Citybikes API and displayed on the screen using the Google Maps library.

Twitch Finder

This app is able to find available live streams on the Twitch service, and list them on your page by most popular streamer. If you click on any stream, it will pull up their stream live within the app. It is a great example of showing real-time information for social media services.

It was created using React and CSS. It also utilizes the Twitch API with Oauth, and works Twitch's limitations on API calls per minute.
